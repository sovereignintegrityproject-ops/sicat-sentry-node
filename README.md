# $SICAT: The Sovereign Intersectional AI Trust

**Sensing Bias. Slashing Inequity. Securing the Global South.**

$SICAT is an agent-native protocol built on **Solana** that bridges **Sony's FHIBE** (Fair Human-Centric Image Benchmark) with decentralized finance. We mathematically enforce ethical AI by slashing biased models and redistributing wealth to the Global South.

## 🚀 The Economic Engine
When a model fails an audit, the protocol enforces a **60/35/5 Split**:
- **60%**: Transferred to the **Global South Vault** (Infrastructure grants).
- **35%**: Distributed to **Auditor Rewards** (Node operators).
- **5%**: Retained for **Sentry Operations** (Protocol maintenance).

## 🛠️ Tech Stack
- **Blockchain**: Solana (Anchor Framework)
- **AI Auditor**: Sony FHIBE (Python Bridge)
- **Sentry Engine**: Node.js (Agent-Native Execution)

## 📡 Workflow
1. **Analyze**: `fhibe_connector.py` evaluates model disparity (EOD/DI).
2. **Execute**: `sentry_engine.js` calculates intersectional severity.
3. **Settle**: On-chain slashing via `sicat_protocol` smart contract.
4. **Verify**: Public QR-code receipt generation via `certification_check.js`.
